About

The Event Plugin is a module federation plugin for the TopLocs platform that enables communities to create, manage, and participate in events. It integrates seamlessly with TopLocs' decentralized P2P architecture.

Integration

To use this plugin in your TopLocs instance, add it to your plugin configuration:

{ "id": "event_plugin", "name": "Events", "url": "https://toplocs.github.io/event-plugin/assets/plugin.js", "version": "1.0.0", "description": "Create and manage events within TopLocs communities", "author": "TopLocs Team" }

The plugin exposes components for the following slots:

  • Info/Sidebar - Event list sidebar widget
  • Info/Content - Main event management interface
  • Settings/Content - Event plugin settings

Plugin Endpoints

The plugin is served via GitHub Pages with the following key endpoints:

  • Plugin Entry: https://toplocs.github.io/event-plugin/assets/plugin.js
  • Landing Page: https://toplocs.github.io/event-plugin/

Development

The Event Plugin is built with:

  • Vue 3 + TypeScript
  • Vite + Module Federation
  • Gun.js for P2P data synchronization
  • Tailwind CSS for styling

To run locally:

# Clone the repository git clone https://github.com/toplocs/event-plugin.git cd event-plugin # Install dependencies pnpm install # Run development server pnpm dev
